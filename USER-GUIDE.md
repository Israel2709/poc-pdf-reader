# Guía de uso

## Ver PDFs

1. Ve a la vista **PDF's**.
2. Usa el campo de búsqueda para filtrar por título.
3. Haz clic en **Ver PDF** para abrir el visor interno.
4. Usa **Anterior** / **Siguiente** para navegar páginas.
5. Opcional: activa **Pantalla completa** para ver mejor el contenido.
6. Cierra el visor con **Cerrar** o haciendo clic fuera del modal.

## Subir un PDF

1. Ve a la vista **Subir PDF**.
2. Arrastra tu PDF a la zona indicada o haz clic para seleccionar un archivo.
3. Revisa o edita el **Título del PDF** (se autocompleta si el PDF tiene metadata).
4. Escribe una **Descripción** si lo deseas.
5. Presiona **Subir PDF** y espera a que finalice.
6. Al terminar, verás un enlace para abrir el PDF subido.

## Cambiar archivo

- Para reemplazar el PDF seleccionado, arrastra otro archivo o usa el botón **Quitar archivo**.
